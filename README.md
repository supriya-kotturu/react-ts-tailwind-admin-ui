# Admin dashboard

## Getting Started

- Clone the repo
  ```
  git clone https://github.com/supriya-kotturu/react-ts-tailwind-admin-ui.git
  ```
- Install the dependencies
  ```
  npm install
  ```
- Build the files with vite
  ```
  npm run build
  ```
- Locally preview the production build
  ```
  npm run preview
  ```
